Strategy Logic: The equity curve is generated by compounding daily returns, which are conditionally modified. If the previous day's closing price was above its 120-day SMA, the actual daily return is used. If it was below or equal, the return is inverted. The smoothed Inverse Inefficiency Index (SMA) from the previous day acts as a multiplier for this (potentially modified) daily return. The background shading indicates the condition applied (green for price above SMA, red for below or equal).
